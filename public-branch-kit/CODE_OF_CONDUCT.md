# Code of Conduct

## Our Pledge

We want Doma to be a welcoming project for people who care about thoughtful, privacy-first software.

Everyone participating in this project is expected to communicate with respect, patience, and good intent.

## Expected Behavior

- Be kind and constructive.
- Assume good intent, but accept corrections gracefully.
- Keep discussions focused on the project.
- Respect different languages, cultures, and family contexts.
- Give feedback on code, design, and documentation without personal attacks.

## Unacceptable Behavior

- Harassment, threats, insults, or discrimination.
- Publishing another person's private information.
- Sexualized language or imagery.
- Repeated off-topic disruption.
- Deliberately submitting malicious code or harmful instructions.

## Privacy Expectations

Doma is a family planning app. Do not ask contributors to share real family data, private schedules, personal documents, or sensitive screenshots.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or block users who violate this code of conduct.

If you need to report a concern, contact the maintainer through GitHub.
